# 📱 INSTALAR LIBRERÍA QR SCANNER

## 📦 INSTALACIÓN

Ejecuta este comando en la terminal:

```bash
npm install react-qr-scanner
```

O si usas yarn:

```bash
yarn add react-qr-scanner
```

## ✅ VERIFICAR INSTALACIÓN

```bash
npm list react-qr-scanner
```

Debería mostrar:
```
react-qr-scanner@X.X.X
```

**¡Ejecuta el comando y luego continúa!** 📦✅
